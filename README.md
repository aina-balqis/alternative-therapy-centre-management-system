# Alternative Therapy Centre Management System (ATCMS)

This project is my Final Year Project developed as part of the Bachelor of Computer Science (Software Engineering) program.

## Project Overview
ATCMS is a web-based management system designed to support the operations of an alternative therapy centre, including appointment booking, payment processing, and reporting.

## Key Features
- Appointment booking with slot validation and conflict prevention
- Online payment integration with callback and fallback handling
- Role-based access control (Admin, Therapist, Client)
- Dynamic reporting based on date and status
- MVC architecture with DAO pattern

## Tech Stack
- Java (Servlets, JSP)
- MySQL
- HTML, CSS, JavaScript

## Notes
This project focuses on system design, backend logic, and web application workflows.
